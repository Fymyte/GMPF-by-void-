# GMPF by void*
Official repo for GMPF project.

Le projet GMPF est un logiciel de retouche d'images. 

- Accéder au dossier GMPF\_project. (cd GMPF\_project depuis un terminal)
- Saisir la commande make
- Saisir ./GMPF.out
